# Power BI Netflix Analysis Dashboard

## Project Overview
This project analyzes Netflix Movies and TV Shows data using Power BI.
The dashboard provides insights into content distribution, duration, ratings,
release years, and country-wise availability.

## Tools Used
- Power BI
- Power Query
- DAX
- Netflix Dataset (CSV)

## DAX Measures Used
- Total Titles
- Movies Count
- TV Shows Count
- Average Duration

## Conclusion
This dashboard helps understand Netflix content trends and supports data-driven
decision-making using interactive Power BI visuals.
